# Skin Lesion Classification using CNN

## Project Description
This project classifies skin lesion images into two categories:
- Benign
- Malignant

A Convolutional Neural Network (CNN) is trained on the HAM10000 dataset with data augmentation to classify skin lesions.

## Dataset
- HAM10000 Dataset

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Project Workflow
1. Data Loading
2. Data Preprocessing
3. Data Augmentation
4. CNN Model Building
5. Model Training
6. Model Evaluation
7. Confusion Matrix
8. Classification Report
9. Prediction

## Results
- Binary Classification (Benign vs Malignant)
- CNN-based image classification
- Model evaluated using Accuracy, Confusion Matrix, and Classification Report
